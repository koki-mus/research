# research
はじめに ".env" というファイルを作り

bigendian_data={データの絶対パス}
imgout={データの出力先絶対パス}
root_dir={pythonプログラムのルートにしたいパス}
snaps_dir={スナップショットのパス}

ex)
imgout=C:/research/imgout/
bigendian_data=C:/research/bigendian_data/
root_dir=C:/research/
snaps_dir=C:/research/Snapshots/
を記述して下さい
